# 🎮 Hangman with Emojis

## Objective

The purpose of this project is to create a visually engaging, emoji-enhanced Hangman game with multiple word categories and a colorful interface.

## Features

* Word categories
* Emoji-based hangman stages
* Hint system
* Already guessed letters tracker
* Visual display of word with underscores
* Score system
* Sound effects

## Tools & Technology

**Language:** Python/Java

**IDE:** VS Code, PyCharm, Thonny, or any text editor

**Database:** Not Required

**Libraries:** tkinter, random

## Expected Outcome

The hangman game picks a random word and shows blank spaces for each letter. Players guess letters one by one, and each wrong guess changes the emoji from happy to sad to dead. When the player wins or loses, the game displays the correct answer and provides an option to play again.
